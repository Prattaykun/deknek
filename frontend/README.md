# Frontend - Internship Portal

Modern authentication frontend built with Next.js (App Router), TypeScript, Tailwind CSS v4, and shadcn-style reusable components.

## Features

- Animated and responsive professional UI
- Login and signup forms
- JWT token persistence in local storage
- Protected profile view loaded via backend token auth

## Setup

```bash
npm install
npm run dev
```

Frontend URL: `http://localhost:3000`

## Environment

Create `.env.local`:

```bash
NEXT_PUBLIC_API_BASE_URL=http://localhost:8000
```

## Build

```bash
npm run build
npm start
```
